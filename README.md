# TDD
Test Driven Development, 테스트코드를 개발 전에 먼저 작성하는 개발 방식
## 그냥 테스트와 다른점
## TDD cicle of life
- 실패하는 테스트를 작성한다 (Red)
- 테스트를 통과하는 로직을 구현한다 (Green)
- 구현한 코드를 리팩토링한다 (Refactor)

모든 요구조건을 만족할 때까지 위 cycle을 반복한다
